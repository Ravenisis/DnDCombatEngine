## Project Use

SRD-derived material in this repository should be adapted into structured data,
models, tests, and rules-resolution workflows rather than copied as large blocks
of reference text. Any committed SRD-derived data or generated rules tables must
preserve attribution, link to the license, and note meaningful changes.

The project does not depend on a developer-specific local SRD checkout for
license compliance. Contributors should use this file as the repo-local
attribution and license reference for SRD-informed work.
